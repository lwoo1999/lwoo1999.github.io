# Liang Wu — Academic Website

Source code for my personal academic website, hosted via **GitHub Pages** at [lwoo1999.github.io](https://lwoo1999.github.io).

## Tech

- Pure HTML, CSS, JavaScript — no frameworks
- Bilingual (English default / Chinese) with one-click toggle
- Responsive, mobile-friendly, print-friendly

## Structure

```
index.html          → Single-page site
css/style.css       → Styles
js/i18n.js          → Chinese/English translations
js/main.js          → Language toggle, smooth scroll, nav highlight, mobile menu
images/photo.jpg    → Profile photo (add yours)
files/cv.pdf        → Academic CV (add yours)
```

## Setup

1. Add your profile photo → `images/photo.jpg`
2. Add your CV PDF → `files/cv.pdf`
3. Deploy: push to `lwoo1999/lwoo1999.github.io` → enable GitHub Pages
